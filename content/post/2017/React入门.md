# 安装

你可以进行一起全局安装

```
sudo npm install -g create-react-app
```

# 创建一个应用

为了创建一个App，执行
```
create-react-app react_demo
cd react_demo
```
执行完毕后，他会在当前目录下创建一个名为 react_demo 的文件夹。

如图所示

![react_01](/media/images/2017/react_01.png)



# 运行

运行该App，可以执行
```
cd react_demo
npm start
```
然后打开浏览器即可以看到自动初始化的界面
